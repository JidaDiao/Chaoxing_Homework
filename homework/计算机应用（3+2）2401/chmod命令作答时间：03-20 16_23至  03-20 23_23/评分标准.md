### 整体评分说明
总分100分,共5道题,每题20分。评分主要依据学生对Linux命令的理解和掌握程度,包括命令的正确性、参数的准确性以及语法的规范性。

### 各题评分细则

**题目1: 查看当前系统的 umask 值 (20分)**
- **(20分)** 正确输入`umask`命令,并上传了包含该命令及其清晰输出结果的截图。
- **(18分)** 截图正确,但包含大量无关命令干扰判断。
- **(10分)** 只提供了正确的文本命令`umask`,但未按要求上传截图。
- **(0分)** 命令错误或未作答。

**题目2: 使用 umask 调整默认权限 (20分)**
- **(20分)** 完全正确,输入`umask 002`。
- **(10分)** umask值计算错误,但基本理解题意和umask的用途（如`umask 006`, `umask 020`等）。
- **(5分)** umask值完全错误,无法体现对题意的理解（如`umask 550`）,或答案中包含无关命令。
- **(2分)** 尝试使用符号表示法但格式完全错误（如`umask -rw-x`）。
- **(0分)** 命令完全错误或未作答。

**题目3: 创建新文件 (20分)**
- **(20分)** 完全正确,输入`touch testfile`或`touch ~/testfile`。
- **(15分)** 命令主体正确,但存在拼写错误（如`touck`, `toucch`）。
- **(15分)** 使用了其他可以创建空文件的命令（如`> testfile`）,因为本题意在考察`touch`命令。
- **(10分)** 创建了文件,但文件名与题目要求不符。
- **(0分)** 命令完全错误或未作答。

**题目4: 设置文件不可删除状态 (20分)**
- **(20分)** 完全正确,输入`chattr +i testfile`。
- **(15分)** 命令主体和参数正确,但文件名存在拼写错误（如`tsetfile`）。
- **(10分)** 命令主体正确,但使用了与题目无关的文件名（如`attrtest`）。
- **(10分)** 命令主体正确,但参数或命令格式错误（如`+itestfile`或`testfile +i`等）。
- **(0分)** 命令完全错误或未作答。

**题目5: 取消文件不可删除状态 (20分)**
- **(20分)** 完全正确,输入`chattr -i testfile`。
- **(15分)** 命令主体和参数正确,但存在拼写错误（如`suodo`）。
- **(15分)** 命令主体和参数正确,但文件名存在拼写错误。
- **(10分)** 命令主体正确,但使用了与题目无关的文件名（如`attrtest`）。
- **(10分)** 命令主体正确,但参数或命令格式错误。
- **(0分)** 命令完全错误或未作答。

**备注:**
- 在root环境下,使用`sudo`命令虽非必要,但功能正确,不予扣分。
- 若学生在命令前加`#`号,视为注释,按其后的命令内容正常评分。
